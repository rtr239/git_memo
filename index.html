GIT Memo

This is a simple memo of worflow in order to use GIT repository.

I) simple workflow to push on Github

  1. git init
  2. git status
  3. git add file_name
  4. git add .
  5. git commit -m "message"
  6. git log
  7. git push

II) In case you need to go back to previous commit

  1. git log
  2. git checkout [commit hash]
  3. git branch
  4. git checkout master (to go back to master)

III) To make a new branch then merge

  1. git branch [new_name]
  2. git merge [new_name]

IV) Pushing to Github

  1. Login Github
  2. Create new repository
  3. Add the remote repository to local (copy the "add origin").
  4. git remote [add origin]
  5. git push -u origin master
  6. git push
